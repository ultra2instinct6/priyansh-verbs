# 🦖 Verb Time! Kids

A simple, kid-friendly web app to learn the **top 100 English verbs** in three tenses: **past, present, and future**.

## How to use
1. Open `index.html` in any web browser (double-click it).
2. Three modes:
   - **📚 Learn** — Flip through verbs one by one with example sentences.
   - **🎯 Quiz** — Pick the correct tense form. Earn ⭐ stars (saved in your browser).
   - **📜 All Verbs** — See the full list at a glance.

## What it teaches
For each verb, the child sees:
- The base (present) form
- The past form
- The future form (`will + verb`)
- An example sentence in each tense
- A friendly emoji to make it memorable

## Files
- `index.html` — page structure
- `styles.css` — kid-friendly styling
- `verbs.js` — the list of 100 verbs and example sentences
- `app.js` — Learn / Quiz / List logic

No build step. No internet needed. Just open and go.
